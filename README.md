# OpsPanel

## **OpsPanel** ##

OpsPanel is a responsive, data-driven SaaS dashboard application designed to manage users, billing, analytics, logs, and notifications. Built with full-stack web technologies, it provides interactive dashboards, KPI tracking, and user management tools for enterprises.

**Features**

**Dashboard**

- KPI cards for quick insights (Total Users, Active Sessions, Transactions, Errors)

- Interactive charts and tables with Chart.js

**User Management**

- Add, edit, delete users

- Role and status management

- Search, filter, and bulk actions

- Persistent data storage with LocalStorage

**Billing**

- View and filter transactions

- Analytics by plan and module

**Logs & Audit**

- View application logs and alerts

- Filter by user, module, and date

- Export logs to CSV

**Analytics**

- Transactions over time

- Users per module

- Revenue breakdown by plan

- Fully responsive UI with modular CSS

**Tech Stack**

**Front-End**: HTML5, CSS3, JavaScript, Chart.js

**Back-End** : ASP.NET Core Minimal API (serving static files)

**Data Persistence**: LocalStorage (browser-side)

**Tools**: Visual Studio, VS Code
